## ADDED Requirements

### Requirement: 项目概述文档生成
系统 SHALL 生成一份项目概述文档，包含项目名称、技术架构、核心功能模块的描述。

#### Scenario: 生成项目概述
- **WHEN** 执行文档生成任务
- **THEN** 输出包含以下内容的 Markdown 文档：
  - 项目名称和简介
  - 技术栈列表（后端框架、AI 框架、数据库等）
  - 核心模块说明
  - 项目目录结构

### Requirement: 技术架构说明
系统 SHALL 包含技术架构说明，描述各技术组件的选择理由。

#### Scenario: 技术架构说明
- **WHEN** 生成技术架构部分
- **THEN** 输出包含以下内容的文档：
  - 后端技术栈及版本
  - 前端技术栈及版本
  - AI 模型接入说明
  - 关键依赖列表

### Requirement: 项目结构展示
系统 SHALL 提供项目目录结构说明。

#### Scenario: 目录结构展示
- **WHEN** 生成目录结构
- **THEN** 输出树形结构展示，包含：
  - 后端源码目录
  - 前端源码目录
  - 资源配置目录
